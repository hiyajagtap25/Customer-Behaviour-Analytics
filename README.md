# Customer-Behaviour-Analytics
This is a data Analytics project that analyzes user behaviour of real ecommerce platforms and generates AI Insights for businesses that use tools like Google Analytics &amp; Microsoft Clarity to track user behaviour.
AI-Powered E-commerce Funnel & Conversion Intelligence
Project Overview
Modern e-commerce businesses generate large volumes of behavioral data, but converting this data into actionable business insights remains a challenge. The objective of this project was to build an end-to-end analytics platform capable of identifying user behavior patterns, measuring conversion performance, and generating AI-powered business insights.
The platform analyzes customer journeys from initial website visits to purchase completion, identifies drop-off points in the conversion funnel, evaluates traffic source quality, and provides automated recommendations using Large Language Models (LLMs).
The final solution was deployed as a live web application using Flask and Render.

Business Problem
E-commerce teams often struggle to answer questions such as:
Where do customers abandon their purchase journey?
Which marketing channels bring high-quality traffic?
Do mobile users behave differently from desktop users?
Which pages contribute most to conversion success or failure?
How can these insights be communicated to non-technical stakeholders?
Traditional dashboards display metrics but require manual interpretation. This project combines analytics with AI-generated insights to reduce the gap between data and decision-making.
